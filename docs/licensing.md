# Licensing intent

The repository includes the full canonical GPL version 3 text in `LICENSE`.
Project source and manifests declare `GPL-3.0-or-later`; contributors grant
permission under GPLv3 or, at the recipient's option, any later version
published by the Free Software Foundation. This intent does not complete the
separate Qt, patent, and distribution review recorded in `LEGAL_GATE.md`.

Do not remove SPDX identifiers, notices, or the full license text when adding a
new language/toolchain. Add its package-level licensing metadata and SBOM input
at the same time.
