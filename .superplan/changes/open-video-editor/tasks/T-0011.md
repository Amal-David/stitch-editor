## Description
Use the fixed fixtures to prove the thin Media Foundation/D3D11 bridge can retain decoded YUV surfaces through a same-device D3D11 color/effect pass, owned RGBA texture, bridge-owned frame lease, and Qt render-thread presentation with correct COM, device, color, synchronization, cancellation, and lifetime semantics.

## Acceptance Criteria
- [ ] Capability probing reports the selected hardware or software MFT, codec profile, native pixel format, color metadata, D3D feature level, and failure reason.
- [ ] Media Foundation NV12/P010 decoder output on the Qt-shared `IMFDXGIDeviceManager` device reaches a validated YUV/range/transfer-to-linear-working-RGBA D3D11 pass, an owned 2D RGBA texture, and Qt render-thread import with zero steady-state GPU-to-CPU frame transfers after warm-up.
- [ ] COM apartments, frame-lease/device-generation ownership, shared-resource synchronization, GPU completion retirement, device loss/recreate, rapid seeks, multi-monitor/resize/DPI changes, project close, and repeated teardown produce no stale frame, deadlock, use-after-free, or leak.
- [ ] QHD-I original and 3K-I proxy playback meet the vertical-slice gates; CPU-oracle samples verify Rec.709 primaries, matrix, limited/full range, transfer, black/white, and source order while traces count transfers, allocations, dropped frames, owned budgets, RSS, and VRAM.
- [ ] Unsupported or corrupt media and unavailable hardware MFTs return typed diagnostics and a disclosed fallback without crash or silent backend substitution.

## Verification

- Run the full Windows development fixture corpus with D3D debug validation, Application Verifier/sanitizers where applicable, device-loss/recreate injection, same-device/backend assertions, and 100 seek-cancellation cycles; never call a buffer-lock path on steady preview frames.
- Inspect GPU/copy traces and retain a failing trace if any readback or lifetime violation occurs.
- Record the exact PC, Windows build, GPU driver, selected MFT, D3D feature level, power/thermal state, and raw samples.
