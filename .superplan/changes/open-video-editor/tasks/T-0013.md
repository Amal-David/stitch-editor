## Description
Integrate the deterministic project compiler, bridge-owned native frame leases and YUV-to-RGBA paths, proven media-I/O adapters, audio graph, bounded scheduler, proxy/cache policy, and one typed semantic render graph shared by preview and export. Implement only the slice operations and effects defined in the acceptance contract.

## Acceptance Criteria
- [ ] The typed demand-driven graph implements source time mapping, transform, opacity, 12-frame dissolve, video fade, deterministic color matrix, audio gain/keyframes/crossfade, compositing, display, and export sinks.
- [ ] Preview and export for the same revision/range share graph construction, time maps, effect ordering, automation sampling, compositing, color intent, missing-media policy, audio routing, and exact `semantic_plan_digest`/`semantic_mix_digest`; each emits a complete allowed-to-differ `execution_plan_digest`.
- [ ] Revision/cancellation epochs, bridge-owned frame leases, bounded editor-owned CPU-frame/GPU-texture/decoder/I/O tokens, and byte-limited queues prevent stale presentation, unbounded memory, early surface reuse, or orphaned work across 100 superseded seeks.
- [ ] Proxy/cache keys include source identity, provider/version, parameters, upstream keys, color digest, and quality profile; final export resolves originals by default.
- [ ] Mandatory QHD-final and 3K-final AVC/AAC MP4 exports pass the independent frame, audio, timestamp, duration, sync, color, sample-entry, and fast-start oracles on both acceptance OSes; unsupported optional HEVC/AV1 profiles return typed results.

## Verification

- Run the canonical two-video/two-audio-track recipe on macOS and Windows through preview, proxy, save/reopen, recovery, and export.
- Compare CPU reference and GPU outputs using predeclared exact or tolerance-based oracles.
- Inspect traces for semantic/execution/mix identity, adapter rounding, frame-lease completion, cache invalidation, owned queue/resource bounds, transfers, cancellation debt, and export backend disclosure.
