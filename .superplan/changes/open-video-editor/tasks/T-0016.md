## Description
Before shared-graph integration, prove the platform media-I/O adapters can probe and ingest the exact baseline sample entries, build deterministic seek indexes, preserve canonical rational timing through named platform conversions, encode the mandatory QHD/3K AVC/AAC profiles, mux correct interleaved output, cancel safely, and publish exports atomically without FFmpeg or NodeAV.

## Acceptance Criteria
- [ ] Both OS adapters probe and ingest QHD-I, QHD-LGOP, 3K-I, and VFR-AV with exact container/sample-entry/configuration disclosure and typed failures for every undeclared variant.
- [ ] Seek/index tests preserve PTS/DTS ordering, VFR sample durations, long-GOP dependency rules, monotonic adapter ticks, and at-most-half-tick recorded conversion error without accumulating rounded relative durations.
- [ ] Each named acceptance machine produces mandatory QHD and 3072x1728 AVC/H.264 + AAC-LC fast-start MP4 through a disclosed hardware or permitted software path; unsupported 3K export fails the task and reopens ADR-001.
- [ ] Independent parse/decode verifies frame and sample count, DTS/PTS/timebases, duration, AAC delay/padding, sync markers, color metadata, sample entry, and fast-start placement against the locked oracle.
- [ ] Cancellation, encoder/muxer failure, disk-full, and process termination never expose a partial final artifact; publication uses a same-filesystem temporary artifact, verified finalize/flush, and atomic rename with deterministic cleanup/recovery.
- [ ] No production or fallback path links, launches, or transitively introduces FFmpeg, NodeAV, MediaBunny server, or an undisclosed codec binary.

## Verification

- Run cold/warm ingest, 100 seeded seeks, VFR and 10,000-frame rational-adapter tests, mandatory QHD/3K exports, and every cancellation/fault point on macOS and Windows.
- Verify outputs with the independent oracle-lock toolchain rather than the app's demux/mux implementation.
- Preserve raw logs, adapter-error records, traces, artifact hashes, actual codec paths, failed temporary artifacts, and clean-final-directory assertions.
