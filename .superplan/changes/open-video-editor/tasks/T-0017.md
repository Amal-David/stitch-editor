## Description
Run the pinned bootstrap workflow from a clean committed checkout on the declared hosted macOS and Windows runners. Preserve the raw logs and toolchain artifacts, correct workflow-only failures, and accept the cross-platform foundation only when both legs pass without downloads inside the canonical build command. This task owns runner evidence, not editor features.

## Acceptance Criteria
- [x] The committed workflow executes from a clean checkout on `macos-15` and `windows-2022` using only immutable action SHAs and the declared Rust 1.97.0, CMake 3.31.6, and Qt 6.11.1 provisioning inputs.
- [x] Both legs pass policy, formatting, lint, Rust unit-test discovery, CMake configure, Qt shell compilation, and CTest invocation through `./scripts/bootstrap.sh all`.
- [x] The macOS artifact records the selected Xcode/macOS SDK, Qt path, compiler, and Metal build assertion; the Windows artifact records Visual Studio 17.x, Windows SDK 10.0.26100.0, Qt path, compiler, and D3D11 build assertion.
- [x] No workflow step relies on a mutable action tag, undocumented Qt action output, build-time network fetch, software/WARP fallback, or an unrecorded toolchain selection.
- [x] Failures are retained and fixed in the repository before rerun; neither platform is represented as verified from workflow syntax or another operating system's local build.

## Verification
